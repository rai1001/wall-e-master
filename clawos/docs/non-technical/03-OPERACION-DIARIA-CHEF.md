# Operación diaria (modo Chef)

## Objetivo
Gestionar la operación diaria con dos ventanas de envío:
- **día anterior por la tarde**
- **día actual por la mañana**

## Rutina recomendada

## 1) Inicio de turno (mañana)

1. Abre **Inicio**.
2. Revisa estado general.
3. En **Chat de Control**, lanza:
   - "resumen del día"
   - "pendientes por turno"
4. Ejecuta envío de coordinación cocina/sala.

## 2) Cierre previo (tarde anterior)

1. Revisa turnos del día siguiente.
2. Lanza comunicación a cocineros asignados.
3. Envía coordinación con sala.
4. Si hay compras, activar recordatorio a compras.

## 3) Revisión semanal

1. Entra a **Proyectos**.
2. Revisa ejecución semanal (ok/fallos).
3. Reintenta lo fallido.
4. Actualiza ajustes de turnos.

## 4) Revisión mensual

1. Confirma envío de horarios mensuales.
2. Verifica recepción en canales (WA + email).
3. Guarda evidencia (logs/actividad).

## Plantilla de control diario (rápida)

- Estado general: verde / amarillo / rojo
- Envío mañana: hecho / pendiente
- Envío tarde previa: hecho / pendiente
- Compras: enviado / no aplica
- Incidencias: ninguna / detalle

## Regla de oro
Si hay duda operativa, prioriza:
1) mantener servicio,
2) comunicar por WhatsApp,
3) dejar registro en actividad.
