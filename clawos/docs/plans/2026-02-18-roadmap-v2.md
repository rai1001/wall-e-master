# ClawOS Roadmap V2

## Phase 1: Core Backend (Week 1)

Deliverables:
- Middleware server boot and health checks
- OpenClaw WebSocket bridge (`ws://localhost:18789`)
- LanceDB memory schema and ingest/search endpoints
- Embedding adapter (OpenAI and local option)

Exit Criteria:
- Live stream from daemon works
- Memory ingest and retrieval pass integration tests

## Phase 2: Agent Personality and Shared Context (Week 2)

Deliverables:
- Agent factory with `SOUL.md`, `IDENTITY.md`, `CONTEXT_BRIDGE.md`
- Context injector before each response
- Handoff endpoint and shared memory transfer

Exit Criteria:
- Agent spawn creates identity files
- Inter-agent handoff works for same project namespace

## Phase 3: Frontend and Voice (Week 3)

Deliverables:
- Next.js PWA chat shell
- Dashboard (kanban/timeline)
- Discovery feed and memory search
- STT + TTS integration with hands-free mode
- Non-technical onboarding and guided first-run experience

Exit Criteria:
- Mobile PWA flow works end-to-end
- Voice command produces spoken response
- New non-technical user completes first task without docs

## Phase 4: Remote Connectivity and Hardening (Week 4)

Deliverables:
- Tailscale Funnel setup
- Cloudflare Tunnel fallback option
- Auth hardening, rate limits, and observability baseline

Exit Criteria:
- Remote access works without router port forwarding
- Security checklist complete
